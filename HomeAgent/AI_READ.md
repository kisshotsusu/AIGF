# HomeAgent AI Read

## 架构与组件
HomeAgent 是一个基于 Python 的家庭 AI 助手，主要组件包括：
- `agent.py`: 核心代理逻辑，包含任务规划、工具调用和执行循环。
- `app.py`: 应用入口，处理 WebSocket 连接和界面交互。
- `home_modules/`: 包含代码编辑器、命令执行器和多模态处理等子模块。

## 接口与工具
- **cloudmusic_search_and_play**: 用于在网易云音乐客户端搜索并播放指定歌曲。
- **bilibili_open_favorite_video**: 用于在 B 站打开收藏夹视频。
- **ui_list_windows / ui_analyze_window**: 用于获取和分析窗口状态。

## 规则与逻辑
- **任务规划**: 由语义规划模型生成任务计划 (task_plan)，包含 domain, site, handler 等字段。
- **Handler 选择**:
  - 规划模型输出 `query_is_explicit`，表示用户是否明确指定了要搜索或选择的内容。
  - 网易云界面任务统一选择 `model_ui`；是否搜索以及后续动作由模型生成的 `steps` 和实时窗口观察决定。
  - 通用打开/播放指令不构造搜索词；指定内容才保留原始 `query`。
- **执行策略**: 根据 handler 和 domain 自动选择 tool_loop, vision_loop 等策略。
- **原子工具执行**: 执行模型仅组合窗口枚举、启动/激活、画面识别、点击、输入、按键和再次识别，不暴露网易云一键搜索播放工具。
- **工具域隔离**: 执行模型只收到当前计划授权的工具；旧专用调用、其他任务或被改写的目标均在执行前拒绝。

## 当前状态
- “打开网易云播放音乐”不会搜索名为“音乐”的曲目；明确指定歌曲时，搜索框、结果和播放控件都由模型根据实时画面处理。
- 代码任务中的错误媒体工具调用不会产生外部副作用。
